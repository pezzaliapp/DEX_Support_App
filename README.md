> Educational prototype inspired by the DEX vision. Client‑side PWA to explore **low‑frequency radio cosmology** from the **lunar far side**.

---

## EN — Abstract
**DEX‑Edu Support App** includes:
- **21‑cm** educational simulator (global signal vs foregrounds),
- **Array Planner** with qualitative baseline histogram and uv sketch,
- **Resolution indicator** (θ ≈ λ/D_max) and |u|max,
- **Lunar Environment & Quiet Window** visualizer,
- **RFI Shield (cinematic)** with export PNG,
- **Payload** dashboard (toy mass & power budget, JSON export, printable report).

**Purpose:** education, outreach, early concept discussion. Models are simplified by design — no external libraries.

### Quick Equations
- Redshift: *z = 1420 / ν − 1*
- Wavelength: *λ ≈ 300 / ν_MHz*
- Resolution: *θ ≈ λ / D*
- Field of View: *FoV ≈ λ / d*

### Install (GitHub Pages)
Upload folder → enable Pages → open URL → *Install* as PWA.  
After updates, perform **hard reload** (versioned service worker).
